🧭 CityCompass

CityCompass is a location-based web app that helps users find nearby essentials like hospitals, restaurants, cafes, and ATMs on an interactive map.

The app uses live location and OpenStreetMap data to display nearby places along with distance, directions, and quick access through a clean and responsive interface.

Features :
Live location detection
Search cities manually
Nearby hospitals, restaurants, and ATMs
Distance-based sorting
Interactive map markers
Google Maps directions
Dark mode
Save favorite places
Responsive design for mobile and desktop
Tech Stack
HTML
CSS
JavaScript
Leaflet.js
OpenStreetMap
Overpass API
